
    OpenZoom SDK
    MultiScaleImage Flash CS3 Example
    
    
    As Flash CS3 doesn't seem to support external SWC libraries, you'll have
    to download the source code of the TweensyZero and the Tweener animation
    libraries and add them to the source path of the MultiScaleImage.fla file.
    
    
    TweensyZero
    -----------
    http://code.google.com/p/tweensy/
    
    Tweener
